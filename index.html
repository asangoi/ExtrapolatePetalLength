ExtrapolatePetalLength

First choose the Sepcies. Then play with the slider. The slider adjusts the Sepal Width. The server.R code then calculates the Petal Width. A simple lm model is used for extrapolating the Petal width, so all the extrapolated values will fall on the fitted line
